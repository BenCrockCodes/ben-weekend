# assets/

Intentionally empty for now: every texture, particle, sound effect and music
track in NEOVOLT is generated procedurally at runtime (WebGL drawing +
WebAudio synthesis), so the game ships with zero binary assets.

Drop future content into the conventional subfolders as it appears:

- `textures/` — sprite sheets / images
- `music/` — streamed soundtrack files
- `sounds/` — one-shot SFX samples
- `fonts/` — self-hosted webfonts (menus currently use Google Fonts with system fallbacks)
- `particles/` — particle definition files
